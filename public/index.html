API Documentation

Technical reference for developers.

POST /api/screenshot

Request Headers

Header wajib yang harus disertakan dalam setiap request.

Header Name Value Description
Content-Type application/json Format data wajib JSON.

Request Body Parameters

Parameter yang dikirimkan dalam body JSON.

Parameter Type Required Default Description & Validation
url String YES - Target URL website. Wajib diawali dengan https:// atau http://.
width Number NO 1280 Lebar viewport browser dalam pixel. Minimal 100.
height Number NO 720 Tinggi viewport browser dalam pixel.
full_page Boolean NO false Jika true, sistem akan men-scroll ke bawah dan mengambil seluruh halaman website.
device_scale Number NO 1 Kepadatan pixel (DPI). Gunakan 2 atau 3 untuk hasil yang lebih tajam (Retina).

Example Request (JSON)

{
  "url": "https://nekolabs.my.id",
  "width": 1920,
  "height": 1080,
  "full_page": true,
  "device_scale": 2
}

Response Object

Struktur JSON yang dikembalikan jika request berhasil (200 OK).

Key Type Description
success Boolean Status keberhasilan request (selalu true).
image_url String URL langsung ke file gambar (PNG). Valid selama 1 jam.
creator String Watermark response author ("SANN404 FORUM").
